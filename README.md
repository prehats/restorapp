#restorapp
Restorapp is a platform for restaurants. It can be installed on a server (local PC with an internet connection) and run on the server.

On the server, servers can view the following on the control panel: tables, server names, number of tables, menus, data, and metrics.

The Android app connects to the server via QR code or manual configuration. Once connected and logged in, the server can go to the table, take the order, send it to the server, and view the order, comments, the server who took it, and the total on the control panel.

Simple, straightforward, clean, and efficient.
